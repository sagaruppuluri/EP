Comprehensive REST API for managing student records in an educational institution.

Features:

  • Complete CRUD operations

Students

Student management operations

Get all students

Retrieve a paginated list of all students with optional filtering.

Query parameters allow you to:

  • Paginate through results
  • Filter by name (partial match)
  • Filter by minimum CGPA
  • Filter by city
  • Sort results
Authorizations:
bearerAuth
query Parameters
page
integer <int32> >= 0
Default: 0

Page number (0-based)

size
integer <int32> [ 1 .. 100 ]
Default: 20
Example: size=20

Number of items per page

name
string
Example: name=John

Filter by student name (partial match, case-insensitive)

minCgpa
number <double> [ 0 .. 10 ]
Example: minCgpa=7.5

Filter by minimum CGPA

city
string
Example: city=Mumbai

Filter by city

sortBy
string
Default: "studentNumber"
Enum: "studentNumber" "name" "cgpa" "createdDate"

Field to sort by

sortOrder
string
Default: "asc"
Enum: "asc" "desc"

Sort order

Responses

Production server

https://api.university.edu/v1/students

Staging server

https://api-staging.university.edu/v1/students

Development server

http://localhost:8080/v1/students

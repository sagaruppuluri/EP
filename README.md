# Enterprise Programming 

## Software Setup 

1. Install IntelliJ IDEA (community edition)
2. Executing a simple Java application using IntelliJ
```
    a. Open IntelliJ
    b. Select New Project
        i. Name: HelloWorld 
        ii. Choose the location where you want to place the project
        iii. Build system : select Maven
        iv. JDK : choose the jdk 17
            * In case of first time user 
            * Select Download 
            * Version: 17 
            * Vendor: Amazon Corretto 17.0.17
            * Location: choose default 
    g. Choose create 
    h. Open the Main java file and choose Run from the toolbar, you should see the sample file executed.
```
